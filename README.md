\## Description

SnapCal is an Android application that helps users track:



\- Daily calories

\- Water intake

\- Steps (Health Connect)

\- Meal logging

\- Food scanning using OpenFoodFacts API

\- Progress tracking



\## Features



\- Dashboard with stats

\- Health Connect integration

\- Scanner for food

\- Manual meal entry

\- Progress screen

\- Shopping list

\- Navigation using Jetpack Compose



\## Technologies Used



\- Kotlin

\- Jetpack Compose

\- Room Database

\- Health Connect API

\- OpenFoodFacts API
